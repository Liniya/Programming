DDS File Importer

DDS File Importer allows to import documents with DDS file format.

Usage: Go to the menu item Tools > Scripts, and press the option named Import DDS. The file should appear in Krita with the selected format. All changes done to the file have to be stored by using File > Save As, or use the Tools > Scripts > Export as DDS (if you have installed my DDS File Exporter).

Preview DDS files: My recommendation is to use a lightweight program like Pico Pixel (https://pixelandpolygon.com/picopixel-download).

Go to DDS File Importer Github page (https://github.com/esuriddick/Programming/tree/main/Python/Krita/DDS_File_Importer) to get more information.
