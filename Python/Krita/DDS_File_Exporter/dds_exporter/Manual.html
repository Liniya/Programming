DDS File Exporter

DDS File Exporter allows to export documents in DDS file format.

Usage: Once you have a document selected, go to the menu item Tools > Scripts, and press the option named Export as DDS.

Export settings: Inside the plugin folder dds_exporter, there is a file called settings.ini. You may use notepad to change the compression used or whether mip maps are generated, which should be the two most relevant settings. Nevertheless, there are also other options available such as force compatibility with DX9 or DX10.

Preview DDS files: Until Krita allows to import and export DDS files, my recommendation is to use a lightweight program like Pico Pixel (https://pixelandpolygon.com/picopixel-download).

Go to DDS File Exporter Github page (https://github.com/esuriddick/Programming/tree/main/Python/Krita/DDS_File_Exporter) to get more information.
