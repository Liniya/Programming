DDS File Exporter

DDS File Exporter allows to export documents in DDS file format.

Usage: Once you have a document selected, go to the menu item Tools > Scripts, and press the option named Export as DDS.

Preview DDS files: My recommendation is to use a lightweight program like Pico Pixel (https://pixelandpolygon.com/picopixel-download).

Go to DDS File Exporter Github page (https://github.com/esuriddick/Programming/tree/main/Python/Krita/DDS_File_Exporter) to get more information.
