Texture Packing

Texture Packing allows to select several different textures to fill different channels of a single texture (Red, Green, Blue and/or Alpha).

Usage: Go to the menu item Tools > Scripts, and press the option named Texture Packing. Follow the instructions provided and the file to be exported based on your settings will appear in Krita.

Go to Texture Packing Github page (https://github.com/esuriddick/Programming/tree/main/Python/Krita/Texture_Packing) to get more information.
